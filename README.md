# Kirby and the Forgotten Land - Wikipédia

Este projeto é uma página web estática que simula uma entrada da Wikipédia para o jogo "Kirby and the Forgotten Land". Ele foi desenvolvido como parte de um desafio ou curso da DIO (Digital Innovation One), focado em HTML e CSS para criar layouts modernos.

## Visão Geral

A página apresenta informações detalhadas sobre o jogo, incluindo:

-   **Introdução**: Uma visão geral do jogo, desenvolvedora, publicadora e data de lançamento.
-   **Jogabilidade**: Detalhes sobre as mecânicas de jogo, habilidades de Kirby e o novo "Mouthful Mode".
-   **Trailer**: Um vídeo incorporado do trailer oficial do jogo.
-   **Premissa**: A história e o enredo principal do jogo.
-   **Desenvolvimento**: Informações sobre a criação e o lançamento do jogo, incluindo o contexto do 30º aniversário da série Kirby.

A estrutura da página inclui um cabeçalho, menus laterais (esquerdo com links para outros jogos da Nintendo e direito com navegação interna no artigo) e um rodapé.

## Tecnologias Utilizadas

-   **HTML5**: Para a estrutura e conteúdo da página.
-   **CSS3**: Para a estilização e layout (o arquivo `main.css` é referenciado).

## Como Visualizar

Para visualizar este projeto, siga os passos abaixo:

1.  **Clone o repositório** (se este projeto estiver em um repositório Git) ou **baixe os arquivos** para o seu computador.
2.  **Navegue até a pasta raiz** do projeto.
3.  **Abra o arquivo `index.html`** em qualquer navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).

Não é necessário nenhum servidor web ou configuração adicional, pois é um projeto puramente estático.
